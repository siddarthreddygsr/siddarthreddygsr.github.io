R.E.C.O.N

Raspberry Pi Engineered Cluster Over the Network ( R.E.C.O.N as we like to call it ) is a Raspberry Pi 4 Cluster secured with VLAN's on a managed switch. R.E.C.O.N is equipped with 8 nodes and powered by SLURM, docker and kubernetes. We deployed Graphana for monitoring the cluster. ;)
